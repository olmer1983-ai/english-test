# Платформа для тестов по английскому языку

Это веб-приложение для создания и прохождения тестов по английскому языку. Оно предоставляет отдельные интерфейсы для учителей (создание тестов, просмотр результатов) и для учеников (регистрация и прохождение тестов).

## Технологии

- React
- TypeScript
- Tailwind CSS
- Vite
- Progressive Web App (PWA)

## Установка приложения (PWA)

Это приложение является Progressive Web App, что позволяет установить его на ваш телефон или компьютер.

- **На телефоне (Android/iOS):** Откройте сайт в Chrome или Safari, и вы увидите предложение "Добавить на главный экран".
- **На компьютере (Chrome/Edge):** В адресной строке справа появится иконка установки. Нажмите на нее, чтобы установить приложение.

## Запуск проекта

Для запуска проекта на вашем локальном компьютере, следуйте этим шагам.

### 1. Требования

- Установленный [Node.js](https://nodejs.org/) (версия 18.x или выше)
- Пакетный менеджер `npm`

### 2. Установка зависимостей

В корневой папке проекта выполните команду для установки всех необходимых пакетов:

```bash
npm install
```

### 3. Запуск в режиме разработки

Для запуска сервера для разработки с автоматическим обновлением страницы при изменениях в коде, выполните:

```bash
npm run dev
```

После этого откройте браузер и перейдите по адресу, указанному в терминале (обычно `http://localhost:5173`).

## Сборка для продакшена

Когда ваше приложение готово к развертыванию, вы можете создать оптимизированную версию для продакшена:

```bash
npm run build
```

Эта команда создаст папку `dist` в корне проекта, содержащую все статические файлы вашего приложения.

## Развертывание на GitHub Pages

1.  Соберите проект командой `npm run build`.
2.  Загрузите содержимое папки `dist` в ваш репозиторий на GitHub, настроенный для развертывания через GitHub Pages.
